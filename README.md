# Hadrian's Wall Utilities

The web app can be found [here](https://noahbolohan.github.io/hadrians-wall-utilities/).

The Hadrian's Wall Utilities application includes tools to support playing the Hadrian's Wall board game. The score sheet found in this application comes equipped with functionality to submit scores to a personal Google Sheets document. However, submitting to a Google Sheets document requires setting one up for yourself. You can do so by following [this guide](https://github.com/NoahBolohan/hadrians-wall-utilities/blob/main/how_to_submit_scores_to_google_sheets.md).

The Hadrian's Wall Utilities application includes the following features:

- **Neutral deck tracker:** Simulate all of the neutral deck's actions and functionality in a game.
- **Score sheet:** Digital score sheet for games of 1 to 6 players.
